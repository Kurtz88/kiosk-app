  USB로 키오스크 데이터 한 번에 넣기 (UPDATE.bat)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 최초 1회 (키오스크 PC에서)
  1) 이 폴더의 UPDATE.bat 을 메모장으로 엽니다.
  2) KIOSK_APP= 뒤를 실제 설치 경로로 바꿉니다.
     예) set "KIOSK_APP=D:\맛집키오스크\kiosk-restaurant-app"
  3) UPDATE.bat 파일 전체를 USB에 복사해 둡니다.

■ USB에 넣을 것
  • 맛집목록.xlsx   ← 엑셀 일괄등록과 같은 양식 (이 이름 고정)
  • uploads 폴더 (선택) ← 사진 파일(jpg, png 등)만 넣기
      엑셀의 이미지URL/약도URL/메뉴URL 에는
      /uploads/파일이름.jpg 처럼 적어 두면, 같은 이름 파일을 uploads에 넣으면 됩니다.

■ 사용 방법
  1) USB를 키오스크 PC에 꽂습니다.
  2) USB에서 UPDATE.bat 을 더블클릭합니다.
  3) 끝나면 키오스크 브라우저를 새로고침합니다.

■ 참고
  • 엑셀은 "추가"만 합니다. 기존 목록을 지우고 다시 넣으려면 관리자에서 삭제하거나
    별도로 DB를 정리해야 합니다.
  • 서버(npm start)가 켜져 있어도 동작하지만, 가끔 DB 잠금이 나면 서버를 잠시 끄고
    UPDATE.bat 을 다시 실행해 보세요.
  • 이 PC에 Node.js가 설치되어 있어야 합니다. (키오스크 앱을 돌리던 그 PC)
